> A retro-styled, 2-player arcade air hockey game featuring reactive physics and neon aesthetics.



## üéÆ Features

- **Local Multiplayer:** 1v1 split-screen gameplay on a single keyboard.
- **Reactive Physics Engine:** The puck accelerates dynamically based on the paddle's velocity at impact (Smash mechanics).
- **Gap Goal System:** Walls are solid bounce surfaces except for a calculated gap exactly **3x** the player's diameter.
- **Neon Visuals:** High-contrast `Cyan` vs `Magenta` theme on a dark background.
- **Circle-to-Circle Collision:** Precise collision detection for round paddles and pucks.
- **Customizable Matches:** Set your winning score limit in the pre-game menu.
